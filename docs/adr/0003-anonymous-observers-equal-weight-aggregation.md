# Anonymous observers, equal-weight aggregation

Observers are fully anonymous: we store no name and no relationship label for them — an `observer_response` is just its words tied to a subject. The "how others see you" profile is the equal-weight average of each observer's individually-normalized tribe scores (not a pooled bag of words), so one observer who selects more words does not gain more influence. The comparison report unlocks only once at least 3 observers have responded, which both makes the average meaningful and preserves anonymity at the individual level. We chose anonymity over named/accountable feedback because candor is the entire value of the 360 — "the gap is where growth lives" only works if observers answer honestly — and equal-weight averaging over pooling so effort (word count) never becomes influence.
